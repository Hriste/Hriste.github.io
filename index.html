Pulsed Doppler Radar Simulation

Digital Signals Processing Final Project

This radar simulation for Digital Signals processing class RPI 2016. The goal of this project was to create a utility for those learning about radar and doppler shifts to begin to understand how the input parameters afect the waveforms. The focus of this project is on generating the raw doppler shifted, pulsed, radar signals received by the radar, however, demonstration of techniques to process received signals has also been developed.

The simulation and final report can be found HERE.
